## üöÄ Quick Command Reference

### Run Tests
```bash
# Single test
python3 testing_scripts.py api.robinhood.com --authorization --verbose

# All tests with templates
python3 testing_scripts.py api.robinhood.com --all \
  --output results.json \
  --submission-template \
  --verbose

# With bash automation
source robinhood_testing_suite.sh
run_full_assessment "api.robinhood.com" "h1_user" "test@example.com" "user" "pass"
```

### View Results
```bash
# Pretty-print JSON report
cat results.json | python3 -m json.tool

# Find critical findings
grep -i "critical" results.json
```
